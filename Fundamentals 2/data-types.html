JavaScript Data Types

strings Arrays are textual data types. Sequences are written using quotation marks. Double or single trunks can be used.
numbers Used to write integers and decimals. We can do four operations (+, -, *, /) with numbers.
boolean Boolean type can hold only two values, true and false. They are often used in conditional expressions.
undefined If a variable is defined but no value has been assigned, it takes exactly this value.
objects These variables can only hold one thing (string or number).
